# Stamina Calculator Plugin

## 插件简介
`astrbot_plugin_stamina_calculator` 是一个实用的游戏辅助插件，专门用于计算体力恢复时间。该插件支持自定义体力上限（25或50点），并能精确计算出体力完全恢复的具体时间（30分钟恢复1点）。计算结果会直接@通知指令发送者。

## 功能说明
- 计算体力完全恢复的具体时间（精确到分钟）
- 支持25点或50点两种体力上限模式
- 计算结果自动@通知请求用户
- 支持指定当前体力和冷却时间参数
- 自动判断恢复完成时间是"今天"还是"明天"

## 插件流程
1. 用户发送指令 `/火 [当前体力] [冷却时间]`
2. 插件接收指令并解析参数：
   - 验证参数是否为有效数字
   - 检查体力值是否在合理范围内（不超过上限）
3. 计算恢复所需时间：
   - 计算剩余体力点数（上限 - 当前体力）
   - 总恢复时间 = 剩余点数 × 30分钟 + 冷却时间
4. 确定恢复完成时间：
   - 基于当前时间加上总恢复时间
   - 判断完成时间是当天还是次日
5. 格式化输出结果：
   - 包含具体恢复完成时间（小时:分钟）
   - 标明是"今天"还是"明天"
6. 发送结果消息：
   - 使用@功能通知请求用户
   - 返回计算结果消息
